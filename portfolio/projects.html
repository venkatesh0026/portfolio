AI-Powered Real-Time Fake Job Posting Detection

Project Overview

An intelligent system designed to automatically identify and flag fraudulent job postings in real-time. Using advanced NLP techniques and deep learning models (BERT, BiLSTM), this system analyzes job posting text to detect suspicious patterns and language commonly used in recruitment fraud.

Problem Statement

Online job portals face significant challenges with fraudulent postings that exploit job seekers. Manual moderation is time-consuming and prone to human error. This project develops an automated solution for rapid and accurate fraud detection.

My Role & Contributions

  • Designed the NLP pipeline for text preprocessing and feature extraction
  • Implemented BERT transformer models for contextual text understanding
  • Developed BiLSTM neural networks for sequence-based pattern detection
  • Created evaluation metrics and performance analysis framework
  • Researched Online Recruitment Fraud (ORF) patterns and detection methodologies

Technologies Used

Python NLP BERT BiLSTM PyTorch / TensorFlow Scikit-learn Pandas Numpy

Key Outcomes & Metrics

  • Accuracy: High precision in distinguishing legitimate vs. fraudulent postings
  • Scalability: Real-time processing for high-volume job posting streams
  • Explainability: Interpretable model outputs for user trust and compliance

Project Status

Currently in active development. The core NLP and deep learning models are being refined based on fraud pattern analysis. Expected completion includes deployment-ready API and web interface.
